# 🌳 나를기억해(Zettelkasten) 공식 사용설명서

안녕하세요, 대표님! '나를기억해' 앱을 100% 활용하여 강력한 지식 정원을 가꿀 수 있도록 돕는 공식 가이드입니다. 🚀

---

## 📌 1. 기본 사용법

### 메모 생성 및 편집
- **새 메모**: 사이드바 상단의 `+` 버튼이나 에디터의 `+` 버튼을 누르세요. (단축키: `Ctrl + N`)
- **자동 저장**: 입력하는 즉시 실시간으로 저장됩니다. 우측 상단의 '저장됨' 표시를 확인하세요.
- **삭제**: 휴지통 아이콘(`🗑️`)을 누르면 현재 메모를 삭제할 수 있습니다.

### 메모 계층 구조 (Indentation)
- 에디터 상단의 `+` 아이콘 중 **가운데 아이콘**을 누르면, 현재 메모의 '하위 메모'가 생성됩니다.
- 사이드바에서 들여쓰기(`↳`)와 함께 시각적으로 구분되어 체계적인 정리가 가능합니다.

---

## 🧠 2. 제텔카스텐 핵심 기술

### 양방향 링크 (`[[...]]`)
- 본문 작성 중 `[[` 를 입력하면 기존 메모 제목들이 자동완성 팝업으로 나타납니다.
- 연결하고 싶은 메모를 선택하면 `[[메모 제목]]` 형태로 삽입되며, 클릭 시 해당 메모로 즉시 이동합니다.
- **백링크**: 메모 하단에서 '나를 가리키고 있는 다른 메모' 정보도 한눈에 볼 수 있습니다.

### 태그 활용 (`#...`)
- 본문에 `#태그명`을 입력하면 사이드바 태그 클라우드에 자동으로 집계됩니다.
- 태그를 클릭하여 관련 메모들만 빠르게 필터링할 수 있습니다.

---

## 🎨 3. 시각적 & 편의 기능

### 3가지 감성 테마
사이드바 상단의 `🎨` 버튼을 눌러 테마를 전환해 보세요:
- **Light**: 깔끔하고 표준적인 업무용 모드.
- **Dark**: 어두운 밤, 몰입도를 극대화하는 미드나잇 모드.
- **Classic**: 종이 질감과 우아한 글꼴로 일기장 감성을 더한 모드.

### 지식 그래프 뷰
- **연결 확인**: 메모들이 어떻게 연결되어 있는지 거미줄 형태의 그래프로 확인하세요.
- **노드 이동**: 노드를 드래그하여 배치하거나, 클릭하여 해당 메모를 즉시 열 수 있습니다.
- **에디터 내 보기**: 편집 중 우측 상단의 그래프 아이콘(`📈`)을 눌러 분할 화면으로도 볼 수 있습니다.

### 스마트 검색
- 사이드바 상단 검색창에 키워드를 입력하면, **제목과 본문에서 일치하는 부분이 형광펜처럼 하이라이팅**되어 나타납니다.

---

## 💾 4. 데이터 및 미디어 관리

### 이미지 업로드
- 에디터 상단의 사진 아이콘(`🖼️`)을 눌러 이미지를 추가할 수 있습니다.
- 업로드된 이미지는 메모 하단 갤러리에 표시되며, 클릭 시 크게 볼 수 있습니다.

### 데이터 백업 및 복구
- **내보내기 (`📤`)**: 현재 모든 데이터를 `.json` 파일로 PC에 저장합니다.
- **가져오기 (`📥`)**: 백업해둔 파일을 불러와 데이터를 복구합니다.
- **주의**: '내보내기' 후 확인창을 닫으면 보안 및 정리 차원에서 앱이 초기화될 수 있으니 반드시 파일을 먼저 안전하게 저장하세요.
